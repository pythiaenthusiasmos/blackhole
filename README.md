# Blackhole

Interactive Kerr metric photon-ray explorer built with React, TypeScript, and Three.js.

Features:

- Kerr metric controls for mass and spin.
- Ray controls for Boyer-Lindquist starting position and constants of motion.
- Single-ray, 1D cluster, and 2D cluster generation with selectable varied parameters.
- Transparent event horizon and ergosphere rendering.
- Trackball rotation, zoom, and pan in the 3D view.

The ray tracer uses first-order null geodesic equations for Kerr spacetime with a compact explicit integrator intended for visual exploration.
